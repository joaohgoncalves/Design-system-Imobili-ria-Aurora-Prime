# Design-system-Imobili-ria-Aurora-Prime
Criei um Design System para a Aurora Prime, plataforma full-stack imobiliária premium. Documentei cores, tipografia, botões, forms, cards, banners, tabelas e padrões de layout em uma página viva no app, com componentes React reutilizáveis para manter consistência, escala e velocidade no desenvolvimento.
